# WebUI - User Interface Design

## Design Principles

1. **Clarity First**: Price information should be immediately visible
2. **Minimal Clicks**: Common actions (add product, view details) in 1-2 clicks
3. **Real-time Feel**: HTMX for instant feedback without page reloads
4. **Mobile-First**: Responsive design for on-the-go price checking
5. **Progressive Disclosure**: Show summary by default, details on demand

## Accessibility Considerations

1. **Keyboard Navigation**: All actions accessible via keyboard
2. **Screen Reader Labels**: Proper ARIA labels for dynamic content
3. **Focus Management**: Focus moves to new content after HTMX swap
4. **Loading States**: `aria-busy="true"` during requests
5. **Error Messages**: Clear, descriptive error text

## Performance Optimizations

1. **Lazy Loading**: Images load on scroll
2. **Debounced Search**: 300ms delay on search input
3. **Conditional Polling**: Only poll on pages that need updates
4. **Mini Charts**: Use lightweight sparklines, not full Chart.js
5. **Pagination**: Limit initial load to 20 products
